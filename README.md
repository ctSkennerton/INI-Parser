This is a simple parser for files with the following format:

```ini
globalKey = value
; This is a comment
# This is also a comment

[Section]
key = value
key2 = value2

[Another Section]
key = value
key2 = value2
```

Only single line values are currently supported.

Parsing returns a two level dictionary.

```smalltalk
parser := IniReader on: iniReadStream.
data := parser parse.
```
